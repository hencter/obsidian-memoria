# Memoria 🪶

> 浮墨式碎片笔记插件 for Obsidian
> **数据永远是纯 Markdown，自由属于你。**

当前版本：**v1.1.1**

---

## ✨ 特性

### 输入
- 浮墨/Blinko 式快速记录卡片，`Ctrl+Enter` 发送
- 标签联想（自动汇集 Vault 全局标签）
- 图片粘贴 / 拖拽 / 选择，自动入库
- **插入表格**：8×8 网格选行列数，一键生成 md 表格模板
- 双击任意笔记进入编辑模式（带高亮边框 + 取消按钮）

### 浏览
- 浮墨式瀑布流时间线，按天分组
- **月历视图 ↔ 14 周热力图**，侧栏一键切换
- 搜索：全文 / `#标签` 直筛 / 多视图组合
- 检索式：今天 / 本周 / 随机回顾 / 无标签 / 有图片 / 有链接
- 置顶 / 收藏（基于 `#置顶` `#收藏` 保留标签，对 Obsidian 全局可见）

### 卡片
- 朋友圈式九宫格图片（1/2/3/4/5–9 张不同布局）+ Lightbox 大图浏览
- **代码块跟随 Obsidian 主题渲染，右上角悬浮「复制」按钮**
- **表格可横向滚动，宽表格也不会撑破卡片**
- 任务列表可勾选，自动回写 md
- Callout 自适应卡片样式
- 标签胶囊点击即筛选
- 卡片右上 `⋯` 菜单：置顶 / 收藏 / 编辑 / 引用 / 打开原文 / 复制 / 删除
- **删除最后一条 memo 会自动清理空的日期标题，md 文件保持整洁**

### 数据报告（独立标签页）
- 总览四宫格（笔记 / 字数 / 活跃天 / 总跨度）
- 365 天大热力图，左右箭头切换年份
- 月度分布柱状图（跟随年份）
- Top 10 标签奖牌排行 + 标签云
- 24 小时活跃分布
- 有趣的发现（最长连续打卡 / 年同比 / 夜猫子等）

### 移动端
- 抽屉式侧栏（顶部菜单按钮唤起）
- 紧凑布局 + 操作按钮常显（无 hover）

## 📂 文件格式

Memoria 在指定文件夹下维护 `YYYY.md` 文件。v1.0.0 起统一采用「时间独占一行 + 内容缩进」格式，美观且避免块级语法渲染问题：

```markdown
# 2026

## 2026-04-25 周六

- 12:43
  这是今天的第一条想法 #灵感

- 14:20
  又想到了一件事
  可以换行继续写，每一行都缩进 2 空格

- 15:47
  > [!tip] callout / 任务列表 / 标题等块级语法都能正确渲染

- 16:30
  - [ ] 任务列表也 OK
  - [x] 自动勾选回写

- 17:05
  {
  "json": "配置片段也能完好保留"
  }
```

每条 `- HH:MM` 开头是一条独立 memo，停用插件后笔记依然完整可读，可以直接用 Obsidian/任意编辑器查看。

## 🚀 安装

1. 复制三件套到 Vault 的插件目录：
   ```
   <Your Vault>/.obsidian/plugins/memoria/
   ├── main.js
   ├── manifest.json
   └── styles.css
   ```
2. Obsidian 设置 → 第三方插件 → 重新加载已安装插件
3. 启用 **Memoria**
4. 左侧 Ribbon 点击 🪶 羽毛笔图标打开面板

---

## ⌨️ 命令

打开命令面板（`Ctrl+P`）输入 "Memoria" 可见：

| 命令 | 说明 |
|---|---|
| 打开 Memoria 面板 | 主视图 |
| 打开数据报告 | 365 天热力图 + 各种统计 |
| 快速记录（弹窗） | 全屏弹窗速记，建议绑全局快捷键 `Ctrl+Shift+M` |
| 规范化所有笔记格式（修复 md 渲染） | 一次性把历史笔记格式升级为块级语法兼容版 |

---

## ⚙️ 设置

- **笔记文件夹**：默认 `Memoria`
- **图片附件文件夹**：默认 `Memoria/attachments`
- **侧栏标签树显示**：默认关闭（与 Obsidian 自带标签栏避免重复）
- **每次加载条数**：默认 50（瀑布流滚动到底加载更多）

---

## 🛡 数据主权

- 100% 纯 md 文本存储，**不依赖** SQLite / IndexedDB / 专有格式
- 置顶 / 收藏用 `#置顶` `#收藏` 标签实现，对 Obsidian Tag Pane 等其他插件透明可见
- 图片直接保存到 vault，用 `![[xxx.png]]` wikilink 引用
- 任意时刻禁用 Memoria，笔记依然完整可读
