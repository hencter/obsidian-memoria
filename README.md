## 💭 为什么做这个

市面上的碎片笔记工具 —— flomo、Memos、Blinko —— 体验都做得不错，但数据都锁在自家数据库里。你的想法应该只属于你，不该被某家公司的服务器或专有格式扣留。

Memoria 做了一个简单的事：**把浮墨式的体验搬到 Obsidian 里，数据存成最普通的 Markdown**。停用插件那一刻，你的 1000 多条笔记依然是可读的 md 文件。

它是克制的 —— 不做 AI 总结、不做云同步、不做分享到微信。这些事别的工具做得更好，让它们去做。Memoria 只管一件事：**帮你把碎片想法无摩擦地记下来，再翻找得出来**。

## ✨ 核心能力

### 🖊 输入
- 浮墨/Blinko 式输入卡片，`Ctrl+Enter` 发送
- 标签联想（自动汇集 Vault 全局所有标签）
- 图片：粘贴 / 拖拽 / 文件选择，自动入库
- 工具栏：`#`标签 · 🖼图片 · ≡无序列表 · 1.有序列表（自增序号） · ☑任务列表 · ▦表格
- **Tab / Shift+Tab** 列表缩进
- **Enter 智能续行**：列表项 Enter 自动加下一条前缀；空列表项 Enter 退出列表
- **输入框自适应高度**：内容多自动撑高，最多 40% 视口高，超出内部滚动
- **草稿持久化**：关闭面板/重启 OB 后，未发送内容自动恢复
- 双击任意卡片进入编辑模式

### 📖 浏览
- 浮墨式瀑布流时间线，按天分组，置顶自动聚集在最上
- 侧栏视图：**14 周热力图** ↔ **月历视图** 一键切换
- 预设视图：全部 / 置顶 / 收藏 / 今天 / 本周 / **每日回顾**（往年的今天）/ 随机回顾（一次 5 条）
- 检索式：无标签 / 有图片 / 有链接
- **搜索组合**：支持 `#标签 关键词` 语法，多标签多关键词 AND 关系
  ```
  #AI 付费           → 有 #AI 标签 且含 "付费"
  #工作 #PUBGM 脑暴   → 同时有两个标签 且含 "脑暴"
  gemini 摸鱼        → 同时含两个关键词
  ```

### 🪪 卡片
- 朋友圈式九宫格图片布局（1/2/3/4/5–9 张不同排列）+ Lightbox 大图（←→/Esc 键盘）
- 代码块跟随 Obsidian 主题渲染，复制按钮悬浮
- 表格可横向滚动，宽表格不撑破卡片
- 任务列表可勾选，自动回写 md
- Callout 自适应卡片样式
- 标签胶囊点击即筛选
- 右上 `⋯` 菜单：置顶 / 收藏 / 编辑 / 引用 / 打开原文 / 复制原文 / 删除
- 删除自动清理空日期标题，md 保持整洁

### 📊 数据报告（独立标签页）
- 总览四宫格：笔记 / 字数 / 活跃天 / 总跨度
- 365 天大热力图 + 年份切换
- 月度分布柱状图
- Top 10 标签奖牌排行（金银铜设计）
- 24 小时活跃分布
- 标签云
- 有趣的发现：最长连续打卡 / 年同比 / 夜猫子 / 最活跃的一天等

### 🛟 安全
- **回收站**：删除时默认保留到 `_trash.md`，误删可恢复（可在设置关闭）
- 所有保留文件（`_` 开头）不进入 Memoria 主视图扫描范围

### 📱 移动端
- 抽屉式侧栏（顶部菜单唤起）
- 紧凑布局 + 操作按钮常显
- 表格选择器改为单击即插入（适配手指）
- 响应式留白，不贴边

## 📂 存储格式

Memoria 在指定文件夹下维护 `YYYY.md` 文件。统一采用「时间独占一行 + 内容缩进」格式，美观且避免块级语法渲染错位：

```markdown
# 2026

## 2026-04-25 周六

- 12:43
  这是今天的第一条想法 #灵感

- 14:20
  又想到了一件事
  可以换行继续写，每一行都缩进 2 空格

- 15:47
  > [!tip] callout / 任务列表 / 标题等块级语法都能正确渲染

- 16:30
  - [ ] 任务列表也 OK
  - [x] 自动勾选回写

- 17:05
  ```json
  { "代码块": "跟随主题" }
  ```
```

每条 `- HH:MM` 开头是一条独立 memo，停用插件后笔记依然完整可读。

## 🚀 安装

### 手动安装（目前唯一方式）

1. 到 [Releases](../../releases) 下载最新版本的 `main.js` / `manifest.json` / `styles.css` 三个文件
2. 复制到 Vault 的插件目录：
   ```
   <Your Vault>/.obsidian/plugins/memoria/
   ├── main.js
   ├── manifest.json
   └── styles.css
   ```
3. Obsidian → 设置 → 第三方插件 → 重新加载已安装插件
4. 启用 **Memoria**
5. 左侧 Ribbon 点击 🪶 羽毛笔图标打开面板

> 若后续提交到官方社区插件库，将支持从"第三方插件市场"直接搜索安装。

## ⌨️ 命令与快捷键

打开命令面板（`Ctrl+P` / `Cmd+P`）搜 "Memoria"：

| 命令 | 默认快捷键 | 说明 |
|---|---|---|
| 打开 Memoria 面板 | — | 主视图 |
| 打开数据报告 | — | 365 天热力图 + 各种统计 |
| 快速记录（弹窗） | **`Ctrl+Shift+M`** / **`Cmd+Shift+M`** | 全屏弹窗速记，不用切到 Memoria 面板 |
| 规范化所有笔记格式 | — | 一次性把历史笔记格式升级为块级语法兼容版 |

## ⚙️ 设置

| 项 | 默认值 | 说明 |
|---|---|---|
| 笔记文件夹 | `Memoria` | `YYYY.md` 存放位置（相对 vault 根） |
| 图片附件文件夹 | `Memoria/attachments` | 粘贴/拖拽/选择的图片保存位置 |
| 在侧栏显示标签树 | 关 | 默认用 Obsidian 自带标签栏，避免重复 |
| 发送后清空输入框 | 开 | — |
| 每次加载条数 | 50 | 瀑布流滚动到底自动加载更多 |
| 删除时保留到回收站 | 开 | 删除的笔记追加到 `_trash.md`，可手动恢复 |

## 🛡 数据主权

- 100% 纯 md 文本存储，**不依赖** SQLite / IndexedDB / 专有格式
- 置顶 / 收藏用 `#置顶` `#收藏` 标签实现，对 Obsidian Tag Pane 等其他插件透明可见
- 图片直接保存到 vault，用 `![[xxx.png]]` wikilink 引用
- 任意时刻禁用 Memoria，笔记依然完整可读
- 零外部依赖（打包产物约 65KB），纯 JS 实现

---

## 🧭 设计原则

Memoria 用这四条原则约束自己"不做什么"：

1. **纯 md，非破坏性存储** —— 卸载插件不丢任何东西
2. **零外部依赖** —— 不引入会膨胀 bundle 的大型库
3. **克制，不越界** —— 碎片笔记做到极致，其他领域（Todo 提醒、AI 总结、云同步、分享图）交给更专业的工具
4. **尊重 Obsidian 生态** —— 标签、附件、快捷键都走 OB 原生机制，不造轮子
